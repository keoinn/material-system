# 物料編碼申請管理系統 V3.5 - 快速啟動指南

## 🚀 5分鐘快速開始

### 步驟 1：開啟系統
直接在瀏覽器開啟 `material_system_v3.5_complete.html`

### 步驟 2：建立第一個物料申請
1. 選擇產品大類：`H - Handle`
2. 選擇產品中類：`01 - Knob`
3. 選擇產品小類：`C - Chrome`
4. 系統自動產生料號：`H01.C.00001`
5. 填寫料件說明
6. 設定包裝（勾選需要的項目）
7. 點擊「提交申請」

### 步驟 3：審核申請
1. 切換到「審核管理」
2. 查看待審核項目
3. 點擊「核准」

### 步驟 4：匯出EXCEL
1. 切換到「EXCEL匯出」
2. 選擇「Handle」類別
3. 點擊「匯出EXCEL」
4. 檔案自動下載

## ⚙️ Python環境設定（Windows）

1. 下載 Python 3.8+ from python.org

## 🆘 需要幫助？

### 快速解答
- **料號產生問題**：檢查大中小類是否都已選擇
- **無法提交**：確認必填欄位（紅色*）都已填寫
- **EXCEL開啟問題**：使用Excel 2016以上版本
