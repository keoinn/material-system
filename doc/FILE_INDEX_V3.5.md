## 🚀 快速開始步驟

### 1️⃣ 基本使用（不需Python）
```
1. 開啟 material_system_v3.5_complete.html
2. 開始使用系統
3. 匯出CSV檔案
```

### 2️⃣ 完整功能（含EXCEL處理）
```
1. 安裝Python 3.8+
2. pip install pandas openpyxl xlrd
3. 開啟 material_system_v3.5_complete.html
4. 使用系統產生申請
5. 執行 python excel_processor.py 處理檔案
```

### EXCEL 輸出格式
- **檔名格式**：`SAP_Material_Export_{類別}_{日期}.csv`
- **編碼**：UTF-8 with BOM
- **分隔符號**：逗號(,)
- **文字限定符**：雙引號(")
