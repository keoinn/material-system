## 💡 使用建議

### 快速開始（3分鐘）

1. **開啟系統**
```
直接開啟 material_system_v3.5_optimized.html
```

2. **測試功能**
```javascript
// 開發者控制台
dev.loadTestData()  // 載入測試資料
dev.showApplications()  // 檢視申請
```

3. **正式使用**
```
1. 物料申請 → 填寫4步驟 → 提交
2. 審核管理 → 核准/退回
3. Excel匯出 → 下載檔案
```

### 部署建議

#### 🏢 企業部署
```
1. Web伺服器部署（IIS/Apache/Nginx）
2. 設定HTTPS安全連線
3. 整合Windows AD認證
4. 連接SAP B1系統
```

#### 🏠 單機使用
```
1. 直接開啟HTML檔案
2. 使用Chrome瀏覽器
3. 定期匯出備份
```

### Python環境（選用）

```bash
# 安裝Python 3.8+
pip install pandas openpyxl xlrd

# 測試Excel處理器
python excel_processor_v35_optimized.py

# 使用範例
python excel_processor_v35_optimized.py convert -i data.json -o output.xlsx
```
